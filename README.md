# PRS - Business Strategy Workspace

PRS is the founder's business strategy and communications workspace for the wider ProjectARC / Deuterium3D ecosystem.

It now holds operating context for:
- Business, CEO, startup, and fundraising material.
- Communications such as emails, scripts, calls, and outreach drafts.
- Strategy for website direction, social direction, networking, and studio positioning.
- Research that informs market, pricing, fundraising, or communications work.
- Partnerships and provisional partner-reference material.
- Historical investor-pitch work that may be mined, but is not live authority.

## Folder map

| Folder | Use |
|---|---|
| `_system/` | Minimal ChatGPT project-source set for PRS. Keep this clean and small. |
| `business/` | Current company narrative, structure, CEO/startup reference, and grounded business context. |
| `strategy/` | Website, social, networking, studio direction, and future communications strategy. |
| `partnerships/` | Partner-specific reference material and provisional terms. |
| `research/` | Market research and source inputs. Re-verify time-sensitive claims before external use. |
| `archive/` | Historical or superseded material. Useful for prior thinking, not live fact. |
| `Human/` | Founder-owned working material. Do not reorganize or rewrite it without explicit instruction. |

## Source-set rule

`_system/` is the default manual upload set for ChatGPT project sources. Working drafts, raw research, historical pitch materials, and `Human/` files are attached only when a task specifically needs them.
